Klasse NumeralSystem

Klasse NumeralSystem: Stellenwertsysteme
Statische Klasse: nur Klassenmethoden.

Übersicht:

Klassenmethoden (statische Methoden)

MethodeBedeutung
binary (n, ascending=False)Binärdarstellung von n als Liste
fromBase (v, b, ascending=False)v aus dem Stellenwertsystem zur Basis b als Zahl
toBase (n, b, ascending=False)n im Stellenwertsystem zur Basis b

Klassenmethoden (statische Methoden)

binary

binary(n)

Aufruf: NumeralSystem.binary(n, ascending=False)

Beschreibung:
Binärdarstellung von n als Liste
Kurzform für toBase(n,2)
Beispiel:
NumeralSystem.binary(11) gibt [1, 0, 1, 1] zurück.
Siehe auch toBase, fromBase.

fromBase

fromBase(v, b)

Aufruf: NumeralSystem.fromBase(v, b, ascending=False)

Beschreibung:
v aus dem Stellenwertsystem zur Basis b als Zahl
v wird als Liste der Stellen erwartet.
Siehe auch toBase, binary.

toBase

toBase(n, b)

Aufruf: NumeralSystem.toBase(n, b, ascending=False)

Beschreibung:
n im Stellenwertsystem zur Basis b
Das Ergebnis wird als Liste der Stellen ausgegeben.
Siehe auch binary, fromBase.
